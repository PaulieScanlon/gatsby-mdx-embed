# PinterestBoard

Display a PinterestBoard by including the component in your `.mdx`. The `pinterestLink` prop is required.

<Props name="PinterestBoard" of={PinterestBoard} />

## Basic usage

<Playground>
  <PinterestBoard pinterestLink="fromspacewithlove/astronauts/" />
</Playground>

## Width

<Playground>
  <PinterestBoard pinterestLink="fromspacewithlove/astronauts/" width={900} />
</Playground>

## Variant

<Playground>
  <PinterestBoard pinterestLink="fromspacewithlove" variant="user" />
</Playground>
