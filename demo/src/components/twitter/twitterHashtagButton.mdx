# TwitterHashtagButton

Display a TwitterHashtagButton by including the component in your `.mdx`. The `hashtag` prop is required.

<Props name="TwitterHashtagButton" of={TwitterHashtagButton} />

## Basic usage

<Playground>
  <TwitterHashtagButton hashtag="#100DaysOfGatsby" />
</Playground>

## Size

<Playground>
  <TwitterHashtagButton hashtag="#100DaysOfGatsby" size="large" />
</Playground>
